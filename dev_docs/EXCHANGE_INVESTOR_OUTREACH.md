# Anonymity Tips

1. **Email Provider:** Use **ProtonMail** (proton.me) or **Tutanota** (tuta.com) — both are end-to-end encrypted and don't require personal info to sign up
2. **Suggested Email:** Something like `core@unauthority.onion` (if you set up a mail server on Tor) or `unauthority-core@proton.me`
3. **Signature:** Always sign as "Unauthority Core Team" — never use personal names
4. **No Social Media Links:** Don't link personal Twitter/LinkedIn. If needed, create an "Unauthority" project account
5. **GitHub:** Your GitHub account (`mky-one`) is already pseudonymous — this is fine
6. **VPN/Tor:** Send outreach emails over **Tor Browser** or a **VPN** to avoid IP logging by email providers
7. **Response Handling:** If exchanges ask "Who is behind this?" — respond with: "Unauthority is an open-source community project. The development team maintains anonymity consistent with the project's privacy-first principles, similar to other prominent blockchain projects."
