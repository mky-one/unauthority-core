WHY 4 NODES MINIMUM FOR PRODUCTION?
  - Byzantine Fault Tolerance: 3f + 1 formula (f = max faulty nodes)
  - For f=1 (tolerate 1 crash): 3(1) + 1 = 4 nodes
  - 1 node = Single point of failure ❌
  - 4 nodes = Can survive 1 node crash ✅

CAN I RUN 1 NODE ONLY?
  ✅ YES for development/testing (testnet)
  ❌ NO for production (mainnet) - network will stop if node crashes
